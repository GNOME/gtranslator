gtranslator is a po-file editing framework consisting of the gtranslator editor, the gtranslatord daemon ( which is for now not quite usable, but this will be the next TODO's ) and the message database features which are resembling the translation memory stuff.

Till now gtranslator can edit the po-files together with you, update them from the sources ( if you have got the sources ), navigate through the translations ( first/previous/next/last message ) and to the next untranslated and fuzzy mesages.
You can also edit the header and change many options in the preferences ( like background/foreground color, font, dot char feature, popup menus etc. ) and use dot char'ed editing like in KBabel.
