What's gtranslator ?

gtranslator is a translation supporting programm for the GNOME desktop environment. It's like KBabel for KDE or other apps who resemble this famous (?) app.
gtranslator is for now only a po-gile editor with some good functions but that will change in the future as it'll get a gtranslatord which'll provide a more comfortable and better translation technique.

News

With this release 0.21 gtranslator now works and it's ready for the IPO&IPR. gtranslator also has got already translations in several different languages.

Future

As mentioned above gtranslator will also incldue a gtranslatord programm ( indeed it does this already but this gtranslatord isn't usefull or really doing something yet. ). This daemon will allow translation team coordination, database building and translation databases building with the use of the open source data format for the open source world : XML. It'll also work over networks ( and of course also in the Internet one day ), so that this daemon would really be something very usefull.
So keep your eyes open for the next releases.
